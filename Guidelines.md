# تطبيق إدارة كروت الإنترنت - دليل التصميم (السوق اليمني)

## مقدمة
هذا التطبيق مصمم خصيصاً للسوق اليمني لإدارة كروت الإنترنت لشبكات MikroTik. يربط بين مالكي الشبكات وبائعي نقاط البيع في جميع أنحاء الجمهورية اليمنية.

## المواصفات اليمنية

### نظام تسجيل الدخول
- **استخدام أرقام الهواتف اليمنية** بدلاً من البريد الإلكتروني
- **أرقام الشبكات المدعومة:**
  - يمن موبايل: 777, 773, 770, 771, 772, 774, 775, 776, 778, 779
  - سبأفون: 733, 734, 735, 736, 737, 738
  - Y (وايمن): 739, 730, 731, 732
  - أرقام إضافية: 780-789
- **تنسيق الأرقام:** 9 أرقام بدون رمز الدولة (+967)
- **مثال:** 777123456، 733987654، 739555000

### العملة
- **استخدام الريال اليمني (ر.ي)** في جميع المعاملات والأسعار
- **تنسيق الأرقام:** استخدام `toLocaleString('ar-YE')` للأرقام الكبيرة
- **أمثلة على الأسعار:**
  - باقة يومية: 12,500 ر.ي
  - باقة أسبوعية: 37,500 ر.ي
  - باقة شهرية: 100,000 ر.ي

### المحافظات اليمنية
يجب استخدام المحافظات اليمنية الـ21 التالية:
- صنعاء، عدن، تعز، الحديدة، إب، ذمار، البيضاء
- أبين، لحج، شبوة، المهرة، سقطرى، حضرموت
- الجوف، مأرب، صعدة، عمران، الضالع، ريمة

### المدن اليمنية
- **صنعاء:** صنعاء، سنحان، أرحب، بني مطر، همدان
- **عدن:** كريتر، المعلا، التواهي، خورمكسر، الشيخ عثمان
- **تعز:** تعز، التربة، المخا، ذباب، المواسط
- **الحديدة:** الحديدة، بيت الفقيه، زبيد، الدريهمي
- **إب:** إب، يريم، جبلة، النادرة، ذي السفال

### أرقام الهواتف
- **التحقق من صحة الرقم:** استخدم `validateYemeniPhone()`
- **التنسيق:** xxx-xxx-xxx
- **الاتجاه:** ltr للأرقام، rtl للنصوص
- **الخط:** font-mono للأرقام

### العملة والأسعار
- **دالة التنسيق:**

- **الألوان:** أخضر للأرباح، برتقالي للمعلق، أحمر للخسائر

## أمثلة على الاستخدام الصحيح

### رقم هاتف يمني
```tsx
const validateYemeniPhone = (phone: string) => {
  if (phone.length !== 9) return false;
  const validPrefixes = ['777', '773', '770', '733', '734', '739'];
  return validPrefixes.includes(phone.substring(0, 3));
};
```
